HOW TO UPLOAD
-------------

1. Clone the repo and create a branch:

     git clone https://github.com/kayrozen/esp-glow.git
     cd esp-glow
     git checkout -b claude/web-console-provisioner-q7k3mz

2. Unzip this archive on top of the clone (overwrites modified files,
   adds new ones):

     unzip -o esp-glow-all-files.zip -d /path/to/esp-glow/

3. Stage, commit, push:

     git add -A
     git commit -m "Web console + browser provisioner + control-event queue

  - Live control layer (MIDI/OSC/web -> cues)
  - Phase 3: web_protocol testable core + web_input scaffold rewrite
  - Phase 1+2: WS client + Preact console UI + mock dev server
  - Phase 5a: browser-based provisioner (WASM port of provision.cpp +
    static editor + Pages CI workflow)
  - Control-event queue: decouple input tasks from render (eliminates
    cross-core data race, validated under ThreadSanitizer)

  make test: 10/10 binaries pass, zero warnings.
  Closes #8."
     git push -u origin claude/web-console-provisioner-q7k3mz

4. Open the PR on GitHub. The PR body is in /home/z/my-project/download/apply-web-console-pr.sh
   (or just write your own — the commit message above is the summary).
